# Evaluation Annotations

Manual ground truth files for dissertation evaluation.

## Team Classification (`team_gt.csv`)

Generated by:
```bash
python -m backend.evaluation.team_classification --sample \
    --tracks path/to/tracks.json --video path/to/video.mp4 \
    --n-samples 200 --output eval_output/team/
```

Open `eval_output/team/team_gt.csv` and fill in `gt_team_id` (0 or 1)
by inspecting the crop images in `eval_output/team/crops/`.

- **0** = team wearing the lighter/first jersey
- **1** = team wearing the darker/second jersey
- Leave blank if ambiguous (e.g. ball in the way, out of frame)

Target: 200 annotated rows takes ~1-2 hours.

## Homography (`homography_gt.csv`)

Using the template in `annotations/homography_gt_template.csv`.

For each row:
1. Open the frame image (extract from video at `frame_idx` using VLC or ffmpeg)
2. Identify the landmark (e.g. "centre_spot", "left_penalty_spot")
3. Record the pixel coordinates (pixel_x, pixel_y) where the landmark appears
4. The pitch_x_cm and pitch_y_cm come from the table below
