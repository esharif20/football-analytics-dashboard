**QUAL-05 / D-09: AutoLoginMiddleware guard**

1. Add a `logging` import at the top of `backend/api/auth.py`:
```python
import logging
```

2. Add a module-level logger after the existing imports:
```python
logger = logging.getLogger("api.auth")
```

3. In the `AutoLoginMiddleware.__init__` method, add a startup warning log when auto-login is active. Change from:
```python
def __init__(self, app: ASGIApp):
    self.app = app
```
To:
```python
def __init__(self, app: ASGIApp):
    self.app = app
    self._dev_mode = settings.LOCAL_DEV_MODE is True
    if self._dev_mode:
        logger.warning(
            "AutoLoginMiddleware is ACTIVE — all requests auto-authenticated as dev user. "
            "Set LOCAL_DEV_MODE=false to disable."
        )
```

4. In the `__call__` method, change the condition from:
```python
if scope["type"] == "http" and settings.LOCAL_DEV_MODE:
```
To:
```python
if scope["type"] == "http" and self._dev_mode:
```

This ensures the middleware checks the cached boolean value (set once at init), and the `is True` comparison in `__init__` ensures only an explicitly truthy value activates auto-login (per D-08, the default is still True, but the check is explicit).
